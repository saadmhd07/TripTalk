# Mobile Responsiveness Checklist

## Common Issues to Fix

### 1. Touch Targets
- [ ] All buttons minimum 44x44px (iOS guideline)
- [ ] Adequate spacing between touch elements

### 2. Text Readability
- [ ] Font sizes appropriate for mobile (min 16px for body)
- [ ] Line height comfortable (1.5-1.6)
- [ ] Adequate contrast

### 3. Layout
- [ ] No horizontal scrolling
- [ ] Proper stacking on narrow screens
- [ ] Navigation accessible
- [ ] Forms easy to fill

### 4. Conversation Screen
- [ ] Messages readable on small screens
- [ ] Textarea doesn't zoom on iOS (font-size >= 16px)
- [ ] Send button always accessible
- [ ] Proper keyboard handling

### 5. AppShell
- [ ] Navigation collapses to hamburger if needed
- [ ] Logo/branding visible
- [ ] User email truncates properly
